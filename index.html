Akses Database Ditolak

Firebase memblokir akses karena aturan keamanan (Security Rules). Anda harus mengizinkan akses di Firebase Console.

// Buka Firebase Console > Firestore Database > Rules

rules_version = '2';

service cloud.firestore {

  match /databases/{database}/documents {

    match /{document=**} {

      allow read, write: if true;

    }

  }

}
